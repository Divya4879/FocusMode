## ⚡ Quick Start Guide

### 1. Installation (30 seconds)
```bash
# Clone the repository
git clone https://github.com/Divya4879/FocusMode.git
cd focusMode

# Load in Chrome
# 1. Open chrome://extensions/
# 2. Enable "Developer mode"
# 3. Click "Load unpacked" → Select folder
# 4. Pin extension to toolbar
```

### 2. First Session Setup
1. **Click Focus Mode icon** in Chrome toolbar
2. **Add trusted sites** to whitelist (Gmail, Slack/Course sites)
3. **Set session duration** (1-2 hrs)
4. **Choose mode**:
   - **Standard**: Easy stop with button click
   - **Strict**: Requires emergency phrase, prevents tampering
5. **Click "Start Focus Session"**

### 3. Strict Mode Configuration
- **Toggle "Strict Mode"** → Emergency phrase field appears
- **Create strong phrase** (8+ characters, mixed case, numbers, symbols)
- **Real-time validation** ensures phrase strength
- **Cannot modify** settings once session starts
