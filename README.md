## ‚ö° Quick Start Guide

### 1. Installation (30 seconds)
```bash
# Clone the repository
git clone https://github.com/yourusername/focus-mode-extension.git
cd focus-mode-extension

# Load in Chrome
# 1. Open chrome://extensions/
# 2. Enable "Developer mode"
# 3. Click "Load unpacked" ‚Üí Select folder
# 4. Pin extension to toolbar
```

### 2. First Session Setup
1. **Click Focus Mode icon** in Chrome toolbar
2. **Add trusted sites** to whitelist (Gmail, Slack, work tools)
3. **Set session duration** (recommended: start with 25 minutes)
4. **Choose mode**:
   - **Standard**: Easy stop with button click
   - **Strict**: Requires emergency phrase, prevents tampering
5. **Click "Start Focus Session"**

### 3. Strict Mode Configuration
- **Toggle "Strict Mode"** ‚Üí Emergency phrase field appears
- **Create strong phrase** (8+ characters, mixed case, numbers, symbols)
- **Real-time validation** ensures phrase strength
- **Cannot modify** settings once session starts
